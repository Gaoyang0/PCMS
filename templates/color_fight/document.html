Introduction

Colorfight is a round based game. For each round, the players can send an action during a period of time, then the server will update based on the actions.

Each player has one kinds of resources:

  • Gold

Players use gold to attack and occupy other cells to collect more gold source and score.

Players use gold to build different buildings on their own cells to help the game process.

At the end of the game, the player with the highest amount of score wins.


Game Feature

MapCell

A MapCell represents a cell that a user can occupy.

The type of MapCell is "Gold" can produce gold for each round.

The game starts with a random number of gold mines.


Building

Players can build new "base" on the MapCell that they has been occupied.


User

Each player enters the game with a username and a game_id.

If a player register with the same username and game_id, it will be treated as the same user.

A player starts with 30 gold and 0 score.


Force Field

A MapCell will have a force_field after it's occupied by a player.

A player can select the occupied MapCell and attack the surrounding MapCell .


Game Flow

Preparation

In the beginning of each round, a map will be generated.

Each cell of the map will have lots of attributes.


Register

A player is allowed to join at the appointed time.

The player will be assigned to a cell, this cell will be the home of the player.


Command

For each round, a player could give a list of valid commands. The possible commands are:

  • attack

  • build


Attack

It will try to attack the cell you specified.

Attack(x, y, cost)

  • x: The abscissa of a map.

  • y: The ordinate of a map.

  • cost: The cost of the offense.


Build

It will try to build a base on the cell you specified.

Build(x, y)

  • x: The abscissa of a map.

  • y: The ordinate of a map.
